Using VS Code effectively

Visual Studio Code (VSCode) is a very powerful tool for various kinds of programmers. However, its wide range of features can be a little overwhelming. Here's a few tips for using VS Code well.

Basic shortcuts

  • Save quickly using Ctrl-S or CMD-S. The white dot in the tab means you have unsaved edits, which won't be shown in the browser. Alternatively, turn on Auto-Save in the File menu.
  • Use Tab and Shift-Tab to add/remove indentation to the current line or group of highlighted lines
  • Highlight some text and use Ctrl-Shift-L or CMD-Shift-L to highlight ALL instances of that text; you can then edit them all. Alternatively, use Ctrl-H or CMD-H to open the (more powerful) Find-Replace tool.
  • Add multiple cursors anywhere with Alt-click or Option+click. You can also Alt-click-drag to highlight multiple places. You can return to a single cursor by clicking anywhere.

Adding Extensions

Extensions allow you to add additional features to VSCode. Click on the "Tetris blocks" symbol on the left to install and manage your Extensions.

There is a large collection of open-source extensions available for installation.

Tips for working with HTML

VS Code has some great built-in features that help with writing/editing HTML.

  • Tags have colored (blue) text; invalid tag names change color to red.
  • If you write a opening tag, closing tags are automatically created.
  • As you work with HTML in VSCode, try to adhere to the standard formatting practices described in the "HTML Basics" page. However, there is also a helpful "Format Document" command in the right-click menu, which attempts to automatically clean up your HTML.

Here are some recommended VSCode Extensions for working with HTML:

  1. "Highlight Matching Tag" (by vincaslt): click on a opening/closing tag, and its corresponding tag will be underlined
  2. "htmltagwrap" (by Brad Gashler): highlight text and press Alt-W to add surrounding tags.
